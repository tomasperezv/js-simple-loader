Class Detail
Loader()
The loader object is in charge of loading the static files defined in the loadDefinition.json file. It uses Ajax requests and dynamic DOM creation of elements for injecting the loaded files, for avoiding blocking. Stores a reference to the pending files to be loaded and when all of them are ready the onSuccess passed to the loadFiles function is executed.
